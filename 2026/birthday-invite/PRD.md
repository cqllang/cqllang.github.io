# 生日邀请函产品需求文档 (PRD)

## 1. 项目概述

### 1.1 项目背景
为庆祝刘浪先生 40 岁生日，需要开发一个简洁美观的生日邀请函网页，方便通过微信分享给亲朋好友。

### 1.2 项目目标
- 创建一个简洁大方的生日邀请函页面
- 支持在微信中打开和分享
- 提供地图导航功能，方便宾客找到活动地点
- 可免费部署到 GitHub Pages 托管

### 1.3 目标用户
- 生日人的亲朋好友
- 主要通过微信群分享触达

---

## 2. 功能需求

### 2.1 核心功能
| 功能 | 描述 | 优先级 |
|------|------|--------|
| 邀请函展示 | 展示生日邀请信息，包括生日人、时间、地点等 | P0 |
| 地图导航 | 点击地址可跳转至地图应用进行导航 | P0 |
| 微信分享 | 支持在微信中打开和分享，显示合适的分享卡片 | P0 |

### 2.2 功能详细说明

#### 2.2.1 邀请函展示
- 页面布局简洁大方，突出重点信息
- 包含生日人姓名、年龄、活动时间、地点等关键信息
- 视觉设计温馨喜庆，符合生日主题

#### 2.2.2 地图导航
- 点击地址区域可跳转至地图应用
- 优先支持腾讯地图或高德地图（微信环境友好）
- 提供准确的经纬度坐标

#### 2.2.3 微信分享
- 页面在微信内置浏览器中正常显示
- 分享时显示合适的标题和描述

## 4. 非功能需求

### 4.1 性能要求
- 页面首屏加载时间 < 3 秒
- 页面总大小 < 1MB（含图片）
- 支持 2G/3G/4G/WiFi 网络环境

### 4.2 兼容性要求
| 平台 | 要求 |
|------|------|
| 微信内置浏览器 | 完全支持 |
| iOS Safari | 完全支持 |
| Android Chrome | 完全支持 |
| 桌面浏览器 | 基本支持 |

### 4.3 响应式设计
- 支持手机端（主要场景）
- 支持平板和桌面端（次要场景）
- 断点建议：768px

### 4.4 部署要求
- 部署平台：GitHub Pages（免费托管）
- 域名：可使用 GitHub 提供的免费域名或自定义域名
- HTTPS：GitHub Pages 默认支持

## 5. 技术方案建议

### 5.1 技术栈
| 类型 | 推荐方案 | 说明 |
|------|----------|------|
| 前端框架 | 纯 HTML/CSS/JS 或 Vue/React | 简单页面可用纯静态实现 |
| 样式方案 | CSS3 或 Tailwind CSS | 根据熟悉度选择 |
| 构建工具 | Vite 或无 | 静态页面可不使用构建工具 |
| 部署 | GitHub Pages | 免费托管 |

### 5.2 地图集成方案
推荐使用腾讯地图或高德地图的 Web SDK：

**腾讯地图**
- 微信环境兼容性好
- 提供地点跳转 API
- 文档：https://lbs.qq.com/

**高德地图**
- 功能完善
- 提供地点跳转 API
- 文档：https://lbs.amap.com/

### 5.3 项目结构建议
```
birthday-invite/
├── index.html          # 主页面
├── css/
│   └── style.css       # 样式文件
├── js/
│   └── main.js         # 交互逻辑
├── images/             # 图片资源
└── README.md           # 项目说明
```
